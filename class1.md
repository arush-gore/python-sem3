# Basics of Python Programming

* Python is a very popular and easy-to-learn prpogramming language, created by Guido van Rossum in 1991.
* Python is a **high-level, interpreted, interactive and object-oriented scripting** language.
* Python is design to be highly readable.
* It is used in a variety of fields, including software development, web development, scientific computing, big data, and Artificial Intelligence.

## History of Python
Invented in the Netherlands, in the eaarly 90s by Guido van Rossum.
Python was conceived in the late 1980s and its implementation was in December 1989.

## Feautures of Python
* **Open Source:** Python is free for everyone to use.
* **Simplicity:** Code is simple and easy to understand.
* **Portability:** Python code can be written in one computer or device can be used in other devices.
* **Large Standard Library**
* **OOPS (Object Oriented Programming):** Break down complex problems into code. Also provides security for real world problems.
* **Easy to Learn**
* **Shows Polymorphism**
* **Dynamic Programming** (not static)
* **Embeddable and Extensible:** Python allows code of other languages such as C, C++ to be embedded into it so that certain functions can be performed, making Python even more powerful.
* **Interpreted:** It is interpreted line-by-line so management of memory and CPU is easy.
* **Library Support:** With the help of libraries such as NumPy, Matplotlib, Scikit-learnsupport the user and allow easy development of code for application or tool.
